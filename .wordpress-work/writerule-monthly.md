# 月別おすすめポイント記事ライティングルール

## 適用範囲

このライティングルールは、「[1〜12月]の浜名湖でおすすめの釣りポイント○選」というタイトルの月別まとめ記事に適用されます。

## 記事構造の基本ルール

### 1. フロントマター

#### 必須項目
```yaml
---
title: [月]の浜名湖でおすすめの釣りポイント[数]選
published: [日付]
description: ""
image: ""
tags: ["[月]釣り", "[主要魚種]釣り", "[季節]釣り", "浜名湖釣り"]
category: まとめ
draft: false
lang: ja
---
```

#### 記載ルール
- **title**: 「[月]の浜名湖でおすすめの釣りポイント[数]選」の形式
- **tags**: 月、主要魚種、季節、浜名湖釣りを含める
- **category**: 必ず「まとめ」とする

### 2. 冒頭部分（導入文）

#### 必須要素
- **月の特徴**: その月の浜名湖釣りの特徴を1〜2文で記載
- **シーズンの説明**: 水温や魚の活性、季節的な特徴を説明
- **記事の目的**: その月に実績のあるポイントを紹介することを明記

#### 記載形式
```
[月]の浜名湖は[季節の特徴やシーズンの説明]。

[水温や魚の活性、季節的な特徴の説明]。

本記事は[季節]の浜名湖で実績のある釣りポイントを厳選してご紹介。[季節の注意点や魅力を記載]。
```

#### 記載ルール
- 月の特徴を簡潔に1〜2文で記載
- 水温や魚の活性について具体的に説明
- 季節的な注意点（天候、気温、混雑など）を記載
- 読者の期待を高める表現を使用

### 3. おすすめポイントリストセクション

#### 見出し形式
```
## [月]のおすすめポイント[数]選
```

#### 記載形式
```
- **[ポイント名](リンクURL)**：[エリア名]

- **[ポイント名](リンクURL)**：[エリア名]

...
```

#### 記載ルール
- ポイント名は太字で強調
- 各ポイントに詳細記事へのリンクを追加
- エリア名（表浜名湖/中浜名湖/奥浜名湖）を明記
- リスト形式で記載
- ポイント数は7〜15選程度（月によって変動）

### 4. 各ポイントの詳細説明

#### 見出し形式
```
### [ポイント名]～[キャッチコピー]
```

#### 記載形式
```
### [ポイント名]～[キャッチコピー]

- **おすすめの対象魚**：[魚種1], [魚種2], [魚種3]

- **おすすめの釣り方**：[釣り方1], [釣り方2], [釣り方3]

- **狙い目の時間帯**：[時間帯1], [時間帯2], [時間帯3]

[ポイントの説明文。その月の特徴、釣り方のコツ、注意事項などを2〜4文で記載。]
```

#### 記載ルール
- **キャッチコピー**: そのポイントの特徴を簡潔に表現（「〜の聖地」「〜の定番スポット」など）
- **おすすめの対象魚**: その月に狙える魚種をカンマ区切りで記載（3〜5種類程度）
- **おすすめの釣り方**: その月に効果的な釣り方をカンマ区切りで記載（2〜4種類程度）
- **狙い目の時間帯**: 効果的な時間帯を記載（「朝・夕マヅメ」「潮止まり前後」など）
- **説明文**: その月の特徴、釣り方のコツ、注意事項を2〜4文で記載
- 各ポイントの説明は簡潔に、その月の特徴を強調

### 5. その月に釣れやすい魚セクション

#### 見出し形式
```
## [月]に釣れやすい魚
```

#### 記載形式
```
- **[魚種名]**：（エサ・ルアー）

- **[魚種名]**：（エサ）

- **[魚種名]**：（ルアー）
```

#### 記載ルール
- その月に釣れる可能性のある魚種をリスト形式で記載
- 釣り方（エサ・ルアー）を括弧内に記載
- 魚種名は太字で強調
- 主要な魚種から順に記載
- 10〜20種類程度を記載

#### 見出し形式
```
## 迷ったらここを見て！[月]の厳選ポイントTOP3！

3. $1
```

#### 記載形式
```

### 2位：[ポイント名]

[ポイント名]は[その月の特徴や魅力を2〜4文で説明]。特に[具体的な特徴や実績]。[読者への推奨やアドバイス]。

### 3位：[ポイント名]

```

#### 記載ルール
- その月に特にオススメのポイントを3つ選出
- 1位、2位、3位の順で記載
- 各ポイントの特徴や実績を具体的に記載
- 読者への推奨やアドバイスを含める
- 各ポイントの説明は2〜4文程度

### 7. まとめセクション

#### 見出し形式
```
## 【まとめ】[月]の浜名湖で[季節の特徴]を楽しもう
## 【まとめ】[月]は浜名湖の[シーズン名]！[特徴を簡潔に]
```

#### 記載形式
```
**[月]の要約**

- [要点1]

- [要点2]

- [要点3]

- [要点4]

- [要点5]

[月]の浜名湖釣りは[季節の特徴や魅力を2〜3文で説明]。

[安全対策や準備について1文で記載]。
```

#### 記載ルール
- その月の要点を箇条書きで5項目程度記載
- 季節の特徴や魅力を2〜3文で説明
- 注意事項やアドバイスを含める
- 安全対策や準備について言及
- ポジティブな表現で締める

## 月別の特徴と記載ポイント

### 1月（厳寒期）
- **特徴**: 根魚（メバル・カサゴ）が活性化、カレイの本格シーズン
- **記載ポイント**: 防寒対策、低活性な魚種の型狙い、混雑が少ない
- **主要魚種**: メバル、カサゴ、カレイ、クロダイ、キビレ、シーバス

### 2月（厳寒期）
- **特徴**: 1月と同様、根魚が中心、カレイシーズン継続
- **記載ポイント**: 防寒対策、風裏ポイントの選択、根魚の穴釣り
- **主要魚種**: メバル、カサゴ、カレイ、クロダイ、キビレ

### 3月（春の始まり）
- **特徴**: 水温上昇で魚の活性が上がり始める、シーズンインの兆し
- **記載ポイント**: 春の兆し、水温上昇の影響、シーズンインの準備
- **主要魚種**: キビレ、シーバス、クロダイ、メバル、カサゴ

### 4月（春本番）
- **特徴**: シーズン本格化、多魚種が狙える、釣り人も増える
- **記載ポイント**: シーズン本格化、混雑の可能性、多魚種の楽しみ
- **主要魚種**: クロダイ、キビレ、シーバス、メジナ、ハゼ

### 5月（初夏）
- **特徴**: 水温上昇、サビキ釣りシーズン開始、多魚種が狙える
- **記載ポイント**: サビキ釣り、多魚種、混雑の増加
- **主要魚種**: クロダイ、キビレ、シーバス、アジ、イワシ、ハゼ

### 6月（梅雨・初夏）
- **特徴**: サビキ釣り本格化、タコ釣りピーク、多魚種の黄金期
- **記載ポイント**: 梅雨の天候、タコ釣り、サビキ釣り、多魚種
- **主要魚種**: クロダイ、キビレ、シーバス、タコ、マゴチ、アジ、イワシ

### 7月（盛夏）
- **特徴**: サビキ釣り最盛期、タコ釣り継続、暑さ対策が必要
- **記載ポイント**: 暑さ対策、サビキ釣り、混雑、早朝・夕方の推奨
- **主要魚種**: クロダイ、キビレ、シーバス、タコ、アジ、イワシ、ハゼ

### 8月（盛夏）
- **特徴**: サビキ釣り最盛期、海水浴シーズンとの競合、暑さ対策
- **記載ポイント**: 海水浴シーズン、暑さ対策、早朝・夕方の推奨
- **主要魚種**: クロダイ、キビレ、シーバス、タコ、アジ、イワシ、ハゼ

### 9月（初秋）
- **特徴**: 水温が適温に、キビレ・シーバスの活性向上、落ちシーズン開始
- **記載ポイント**: 落ちシーズン、大型狙い、適温の魅力
- **主要魚種**: キビレ、シーバス、クロダイ、カレイ、ハゼ、アジ

### 10月（秋本番）
- **特徴**: 黄金シーズン、カレイシーズンイン、大型キビレ・シーバス
- **記載ポイント**: 黄金シーズン、カレイシーズンイン、大型狙い、落ちシーズン
- **主要魚種**: キビレ、シーバス、カレイ、クロダイ、ハゼ、アジ、キス

### 11月（晩秋）
- **特徴**: 落ちシーズン継続、カレイシーズン本格化、大型狙い
- **記載ポイント**: 落ちシーズン、カレイシーズン、大型狙い、防寒対策
- **主要魚種**: キビレ、シーバス、カレイ、クロダイ、メバル

### 12月（初冬）
- **特徴**: 冬のシーズン本格化、根魚の活性化、カレイシーズン継続
- **記載ポイント**: 冬のシーズン、根魚の活性化、防寒対策、風裏ポイント
- **主要魚種**: カレイ、メバル、カサゴ、クロダイ、シーバス、キビレ

## ポイント選出の基準

### 選出基準
1. **その月に実績があるポイント**
2. **その月の特徴的な魚種が狙えるポイント**
3. **エリアバランス**: 表・中・奥のバランスを考慮
4. **釣り方の多様性**: エサ釣り・ルアー釣りのバランス
5. **アクセスの良さ**: 初心者でも行きやすいポイントも含める
6. **季節性**: その月に特に効果的なポイント

### ポイント数の目安
- **冬（12〜2月）**: 9〜10選
- **春（3〜5月）**: 9〜11選
- **夏（6〜8月）**: 10〜15選
- **秋（9〜11月）**: 11〜14選

## 文体とトーン

### 文体の特徴
- **親しみやすい口語調**: 「〜ですね」「〜ですよ」「〜かもしれません」など
- **読者への語りかけ**: 「〜してみてください」「〜してみましょう」など
- **共感を促す表現**: 「〜でしょう」「〜かもしれませんね」など
- **専門用語の説明**: 専門用語を使う場合は簡潔に説明を追加

### トーンの特徴
- **ポジティブ**: ネガティブな情報も前向きに表現
- **具体的**: 曖昧な表現を避け、数値や時期を具体的に記載
- **親切**: 初心者にも分かりやすい説明
- **実用的**: 実際の釣りに役立つ情報を重視

## 表現のルール

### 強調の使い方
- **太字**: 重要なポイント名、魚種名、月名など
- 強調記号: 「**○○**」の形式で使用

### 時間帯の表現
- 「朝・夕マヅメ」「朝マヅメ」「夕マヅメ」
- 「潮止まり前後」「潮止まりからの上げか下げ」
- 「下げ7分、上げ3分」「上げ3下げ7」
- 「夜間」「夜釣り」

### 季節の表現
- 「厳寒期」「冬のシーズン」「冬の釣りシーズン」
- 「春の始まり」「シーズン本格化」「シーズンイン」
- 「梅雨時期」「盛夏」「初秋」「秋本番」「晩秋」
- 「落ちシーズン」「黄金シーズン」

### 魚種名の表記
- 正式名称を使用（「キビレ」「シーバス」など）
- カタカナ表記を基本とする
- 複数魚種はカンマ区切りで記載

## 記事の長さ

### 推奨記事構成
1. 冒頭部分: 3〜5文
2. おすすめポイントリスト: 7〜15項目
3. 各ポイントの詳細説明: 各2〜4文 × ポイント数
4. その月に釣れやすい魚リスト: 10〜20項目
5. 厳選ポイントTOP3: 各2〜4文 × 3
6. まとめ: 5項目の箇条書き + 3〜5文

### 記事の長さ
- 標準的な記事: 200〜300行程度
- ポイント数が多い記事: 300〜400行程度

2. **その月の特徴的な魚種が特に狙えるポイント**
3. **初心者でも楽しめるポイント**
4. **上級者向けのテクニカルなポイント**
5. **エリアバランス**: 表・中・奥からバランスよく選出

### 順位の決め方
- **1位**: その月に最もオススメのポイント（初心者向けまたは総合的に優れている）
- **2位**: その月に特徴的な魚種が特に狙えるポイント
- **3位**: 上級者向けまたはテクニカルなポイント

## 季節別の注意事項

### 冬（12〜2月）
- 防寒対策の重要性
- 風裏ポイントの選択
- 釣り人が少なくなるメリット
- 根魚の活性化

### 春（3〜5月）
- シーズンインの準備
- 水温上昇の影響
- 混雑の可能性
- 多魚種の楽しみ

### 夏（6〜8月）
- 暑さ対策
- 早朝・夕方の推奨
- 梅雨の天候注意
- 海水浴シーズンとの競合
- サビキ釣りの本格化

### 秋（9〜11月）
- 落ちシーズンの大型狙い
- 適温の魅力
- カレイシーズンイン
- 朝晩の冷え込み対策
- 台風の影響

